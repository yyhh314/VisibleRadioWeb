@Scheduled 옵션

@Scheduled(fixedRate = 1000, initialDelay = 2000)
cron	원하는 시간대를 설정하여 작업을 실행한다.
@Scheduled(cron = "* * * * * *")
(초(0-59), 분(0-59), 시간(0-23), 일(1-31), 월(1-12), 요일(1-7, 1:일, 7:토))


* : 모든 값
? : 설정 없음
, : 배열 설정 ex) 1,3,5
- : 범위 설정 ex) 3-5
/ : 특정 값을 기준으로 특정값 마다 반복

0 0 0 * * * : 매일 00시 00분 00초에 실행
0 0 2 * * MON : 월요일 새벽 2시에 실행
0/30 * * * * ? : 30초 마다 실행
